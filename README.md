# Santhosh Portfolio

Single-page portfolio site for **Santhosh Kumar M**.

## Stack
- HTML5
- CSS3
- Vanilla JavaScript

## Local run
Open `index.html` directly in a browser, or use a static server:

```bash
python3 -m http.server 8080
```

Then visit `http://localhost:8080`.

## Deploy (Vercel)
1. Push this folder to GitHub.
2. Import the repo in Vercel.
3. Keep framework preset as `Other`.
4. Deploy.

`vercel.json` is included so `/` serves `index.html`.
